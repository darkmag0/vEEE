This is awesome readme for awesome vEEE

В папке "sources" - классы приложения. Создаём проект, добавляем туда классы. В настройках проекта ставим "Use unsafe code" для работы с указателями.
В папке "bin" - файлы конфигурации, их нужно скинуть в папку с программой. Например, у нас проект в папке C:\Projects\vEEE. Тогда после первой сборки появится папка vEEE\bin\Debug. Этот файл нужно скинуть туда.
